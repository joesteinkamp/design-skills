## Prompt A - FigJam Board Builder

Write a prompt that tells an AI assistant to create the board structure and content.

Include:
- board title and section sequence,
- time boxes for each section,
- exact section instructions to render on canvas,
- sticky-note prompts for each exercise,
- dedicated zones for owners and next steps,
- visual organization constraints (clear labels, participant flow, no orphan frames).

## Prompt B - Facilitator Script

Write a prompt that tells an AI assistant to generate a facilitation run-of-show.

Include:
- opening framing,
- segment-by-segment script with timing,
- transition lines between activities,
- group management guidance (quiet participants, dominant voices, parking lot items),
- prompts for surfacing commitments and ownership.
